Facilis Demo App

Deployment-ready space for future format demos.

This route is deployed with the docs site so Facilis can ship live, framework-agnostic demos alongside its documentation. The runtime demos themselves can land here later without changing the hosting model.

What this gives you now

Stable URL A public `/demos/` surface on GitHub Pages from day one.
Docs alignment Demos live under the same deploy as the reference docs.
Room to grow Static HTML pages, bundled examples, or richer playgrounds can all be added here later.

Return to the Facilis docs home when you are ready to wire in the first real runtime examples.
