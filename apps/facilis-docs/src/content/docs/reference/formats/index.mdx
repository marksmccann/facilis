The primary format API reference lives here. These pages document the current
first-party exports from `facilis-formats`.

## Package

```ts
import { currency, number, pattern } from 'facilis-formats';
```

Use `facilis-formats` when you want ready-made format definitions instead of
building a custom one directly with `defineFormat()`.
